Privacy Policy

Last updated: March 30, 2026

1. Overview

ScreenSteps is a local Chrome extension. It does not collect, transmit or store any personal data on external servers. All data generated during use — recordings, screenshots and settings — remains exclusively on your device inside the Chrome browser's local storage.

2. Data collected and where it is stored

The extension stores the following data locally using chrome.storage.local:

  • Recordings – step descriptions, timestamps and the URL of each recorded page.
  • Screenshots – PNG images of the visible browser tab captured at each step.
  • Settings – your preferred UI language and documentation language.

This data never leaves your device. It is not sent to any server, cloud service, analytics platform or third party — ever.

3. Permissions explained

  • activeTab / tabs – required to identify the currently active tab and capture its title and URL for step descriptions.
  • scripting – required to inject the content script that listens for clicks, inputs and navigations on the page you are recording.
  • storage / unlimitedStorage – required to persist recordings and screenshots locally in chrome.storage.local.
  • downloads – required to trigger ZIP and HTML file downloads when you export a recording.
  • host_permissions (http://*/* and https://*/*) – required so the content script can run on any website the user actively chooses to record. ScreenSteps only activates on a page when you explicitly start a recording. No background monitoring takes place.

4. No tracking, no analytics

ScreenSteps does not use any analytics SDK, crash reporter, A/B testing framework or advertising library. There are no cookies, no fingerprinting and no network requests of any kind originating from the extension.

5. Data sharing

No data is shared with any third party because no data leaves your device. Exports (Markdown, HTML, PDF) are files you create yourself and choose to share; ScreenSteps has no involvement after a file is saved to your Downloads folder.

6. Data retention and deletion

Recordings are stored until you delete them manually via the ScreenSteps manager. Uninstalling the extension from Chrome removes all associated storage automatically.

7. Children's privacy

ScreenSteps does not collect personal information from anyone, including children under the age of 13.

8. Changes to this policy

If this policy changes, the updated version will be published at this URL with a new "Last updated" date. Continued use of the extension after a change constitutes acceptance of the revised policy.

9. Contact

Questions about this privacy policy can be directed via GitHub Issues.
