Built for people who document workflows, build tutorials or write SOPs.

📸

Automatic Screenshots

After every click, form input or page navigation ScreenSteps captures the visible tab — no manual snipping required.

🎯

Visual Annotations

Each screenshot highlights the exact element you interacted with using a coloured dot or rectangle. No guessing where to look.

✏️

Editable Steps

Reorder, rename and toggle steps in the built-in manager before exporting. Delete noise, keep signal.

📦

Four Export Formats

Markdown ZIP, HTML ZIP, Confluence-ready HTML and a print-to-PDF view — one recording, every format.

🌍

German & English

UI language and documentation language are independent settings. Record in English, review in German — or vice versa.

🔒

Fully Local

All data stays in chrome.storage.local on your device. No backend, no analytics, no account.

Three steps. That's it.

From zero to exportable documentation in under a minute.

1

Start Recording

Click the ScreenSteps icon in your toolbar and hit Start. The extension silently watches in the background.

2

Interact Normally

Click buttons, fill forms, navigate pages. Every meaningful action is captured with a screenshot and a readable description.

3

Export & Share

Stop the recording, open the manager, edit if needed and export as Markdown, HTML, Confluence or PDF.

Publish anywhere.

One recording exports to four formats — pick whatever fits your workflow.

📝 Markdown ZIP
🌐 HTML ZIP
🔵 Confluence Export
🖨️ PDF (print view)

Your data stays yours.

ScreenSteps is a purely local extension. It never connects to any server, never stores anything outside your browser, and never tracks anything.
